JavaScript Function - Practical Tasks

1. Create a normal function that returns the sum of two numbers.
2. Write a function expression that returns the product of two numbers.
3. Create a function with default parameters.
4. Demonstrate passing a function as an argument.
5. Create a function that returns another function.
6. Demonstrate function hoisting with a normal function.
7. Show how function expression hoisting differs from normal functions.
8. Write a function that uses multiple parameters and returns a string.
9. Create an anonymous function assigned to a variable.
10. Create a named function expression.
11. Show an example of function overloading using arguments length.
12. Write a function that modifies a global variable.
13. Create a function that calculates the factorial of a number.
14. Write a function that checks if a number is even or odd.
15. Demonstrate passing an object as a function argument.
16. Create a function that returns an object with multiple properties.
17. Write a function that converts a string to uppercase.
18. Demonstrate the use of a function inside an object.
19. Show an example of a recursive function.
20. Create a function that reverses an array.
