JavaScript Functions - Interview Q&A

1. What is a normal function in JavaScript?
A normal function is declared using the `function` keyword and supports hoisting.
2. What is a function expression?
A function expression assigns a function to a variable and is not hoisted.
3. What is the key difference between a normal function and a function expression?
Normal functions are hoisted, whereas function expressions are not.
4. What is the purpose of function parameters?
Parameters act as placeholders in a function to accept values when the function is called.
5. What is the difference between parameters and arguments?
Parameters are placeholders in a function declaration, whereas arguments are actual values passed to the function.
6. What is hoisting in JavaScript?
Hoisting is JavaScriptâ€™s behavior of moving variable and function declarations to the top of their scope before execution.
7. Are function expressions hoisted?
No, function expressions are not hoisted. They must be declared before use.
8. What happens if you call a normal function before its declaration?
Since normal functions are hoisted, they can be called before their declaration without error.
9. What happens if you call a function expression before its declaration?
It results in a ReferenceError because function expressions are not hoisted.
10. Can functions return multiple values?
Yes, by returning an array or an object.
11. What is an anonymous function?
A function without a name, usually used in function expressions.
12. What is a named function expression?
A function expression where the function has a name.
13. What happens if you redeclare a function using `var`?
It will override the previous function declaration.
14. Can you pass a function as an argument to another function?
Yes, JavaScript supports higher-order functions.
15. What is a default parameter?
A parameter with a default value assigned in the function definition.
16. What is a rest parameter?
A syntax to handle multiple arguments as an array (not included per request).
17. Can you return a function from another function?
Yes, JavaScript supports returning functions.
18. What is function overloading?
JavaScript does not support function overloading like other languages.
19. How do you prevent variable hoisting?
Use `let` or `const` instead of `var`.
20. Can a function be assigned to an object property?
Yes, such functions are called methods.
